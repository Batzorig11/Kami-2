Параметр ба буцаах утга (return)

🎯 Зорилго

Функцэд утга дамжуулж (параметр), функцээс утга буцаахыг (return) сурах.

📘 Шинэ ойлголт

Параметр (def f(x):), return, буцаасан утгыг хувьсагчид авах.

💡 Тайлбар

Функцэд оролт өгч болно — үүнийг параметр гэнэ (def ask_number(count): дэх count). Функц мөн хариу буцааж болно — return түлхүүр үг ашиглана. Буцаасан утгыг хувьсагчид авч ашиглана: pick = ask_number(2). Өнөөдөр бид 5-р хичээлийн «зөв тоо асуух» давталтыг ask_number(count) функц болгож, тоглоомын аль ч хэсэгт дахин ашиглаж болохоор болгоно.

✍️ Дасгал (хичээл дээр)

  1. ask_number(count) функц бичих — 1..count хүртэлх зөв дугаар буцаана (5-р хичээлийн давталтыг функц болгох).
  2. Функц True/False буцаадгийг туршиж үзэх.
  3. ask_number-г хэсгүүддээ ашиглах.

💻 Гарах код — namjil.py

namjil.py 
def ask_number(count):
    """1-ээс count хүртэлх ЗӨВ дугаар авах хүртэл асууна."""
    while True:
        answer = ask("  Чиний сонголт (дугаар): ")
        if answer.isdigit():
            number = int(answer)
            if 1 <= number <= count:
                return number
        print(color("  1-ээс " + str(count) + " хүртэлх дугаар бичнэ үү.", "red"))
📝 Компьютергүй сурагчид

«Машин» тоглоом — оролт өгөхөд гаралт буцаадаг хүн-машин. Параметр болон буцаах утгыг биеэр дүрслэх.

✅ Дүгнэлт — өнөөдөр юу сурсан

Параметр = функцэд өгөх оролт. return = функцээс хариу буцаах. ask_number(count) нэг удаа бичээд хаа сайгүй дуудна.
