Python-той танилцах

🎯 Зорилго

Терминал нээж, эхний программаа ажиллуулж, print() болон тэмдэгт мөр (string)-ийг ойлгох.

📘 Шинэ ойлголт

print(), тэмдэгт мөр ("..."), тайлбар (#), программ ажиллуулах (python3 файл.py).

💡 Тайлбар

Энэ бол хамгийн эхний хичээл. Компьютерт юм хэлэхийн тулд бид код бичдэг. print() бол хамгийн анхны тушаал — хаалт дотор бичсэн зүйлийг дэлгэцэнд хэвлэдэг. Хашилт ("...") дотор бичсэн үсэг, өгүүлбэрийг тэмдэгт мөр (string) гэнэ. # тэмдгээр эхэлсэн мөрийг Python үл тоомсорлодог — энэ нь зөвхөн хүн уншихад зориулсан тайлбар. Кодоо файл.py нэрээр хадгалж, терминал дээр python3 файл.py гэж ажиллуулна.

✍️ Дасгал (хичээл дээр)

  1. mengu.py нэртэй файл үүсгэж, өөрийн нэрийг хэвлэх.
  2. Үлгэрийн гарчиг, нэг өгүүлбэрийг хэвлэх.
  3. Тайлбар (#) нэмж, түүнийг Python үл тоомсорлодгийг харах.

💻 Гарах код — mengu.py

mengu.py 
# Энэ бол миний эхний Python программ
print("Сайн уу!")
print("Царцаа Намжилын аялал")
print("Эрт урьд цагт нэг тэнэг хүн байжээ...")
📝 Компьютергүй сурагчид

Самбар дээр print(...) дотор юу бичвэл юу гарахыг таамаглах тоглоом. Багш карт өргөнө, сурагчид «гаралтыг» дуудна.

✅ Дүгнэлт — өнөөдөр юу сурсан

Код = компьютерт өгөх тушаал. print() хэвлэнэ, "..." бол текст, # бол тайлбар. Кодыг python3 файл.py-аар ажиллуулна.
