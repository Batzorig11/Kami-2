SCENES жагсаалт + main давталт

🎯 Зорилго

Хэсгүүдийг өгөгдлийн жагсаалт болгож, main()-ээр дараалан тоглуулах.

📘 Шинэ ойлголт

Толь бичгүүдийн жагсаалт (SCENES = [ {...}, {...} ]), жагсаалтыг for-оор дамжих, player["stage"] шинэчлэх.

💡 Тайлбар

Бид play_scene()-тэй болсон тул хэсгүүдээ зүгээр л өгөгдлийн жагсаалт болгож хадгалахад л болно: SCENES = [ {...}, {...} ]. main() функц нь for-оор бүх хэсгийг дараалан гүйж, тус бүрийг play_scene()-д өгнө. Хэрэв play_scene False буцаавал (тоглогч үхсэн) тоглоомыг зогсооно. Шинэ хэсэг нэмэхэд зөвхөн SCENES-д нэг толь бичиг нэмэхэд л хангалттай!

✍️ Дасгал (хичээл дээр)

  1. 1–4-р хэсгийг (Хийд, Шувуу, Зээр, Хулгана) толь бичиг болгож SCENES-д нэмэх.
  2. main() бичиж бүх хэсгийг дараалан тоглуулах.
  3. play_scene False буцаавал тоглоомыг зогсоох.

💻 Гарах код — namjil.py

namjil.py 
SCENES = [
    { "title": "1-р хэсэг: Хийдэд", ... },
    { "title": "2-р хэсэг: Хар шувуу", ... },
    { "title": "3-р хэсэг: Хоёр зээр", ... },
    { "title": "4-р хэсэг: Хоёр хулгана", ... },
]

def main():
    number = 1
    for scene in SCENES:
        player["stage"] = number
        if not play_scene(scene):
            print("Аялал тасарлаа.")
            return
        number = number + 1
    print("Чи яллаа!")
📝 Компьютергүй сурагчид

Дөрвөн хэсгийн текстийг зохиох баг даалгавар — кодоо бичих сурагчдад «өгөгдөл» болгон өгнө (бичих хүн ба зохиох хүний хамтын ажил).

✅ Дүгнэлт — өнөөдөр юу сурсан

SCENES = хэсгүүдийн өгөгдлийн жагсаалт. main() тэдгээрийг дараалан тоглуулна. Шинэ хэсэг = шинэ толь бичиг нэмэх.
