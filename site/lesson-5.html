while давталт — оролтыг шалгах

🎯 Зорилго

Зөв хариулт авах хүртэл дахин дахин асуухыг сурах.

📘 Шинэ ойлголт

while давталт, давталтаас гарах нөхцөл (break), .isdigit(), int().

💡 Тайлбар

while нөхцөл: гэдэг бол «нөхцөл үнэн байгаа цагт давт» гэсэн утга. while True: бол төгсгөлгүй давталт — гарахын тулд дотор нь break хэрэгтэй. Энэ нь хэрэглэгч зөв зүйл бичих хүртэл дахин дахин асуухад тохиромжтой. .isdigit() нь текст бүхэлдээ цифр мөн эсэхийг шалгана, int() нь текстийг тоо болгож хувиргана.

✍️ Дасгал (хичээл дээр)

  1. Зөв тоо (1, 2, 3) бичих хүртэл дахин асуудаг давталт бичих.
  2. Тоо биш зүйл бичвэл сэрэмжлүүлэг хэвлэх.
  3. Энэ давталтыг 4-р хичээлийн тоглоомд залгах.

💻 Гарах код — асуулт.py

асуулт.py 
while True:
    answer = input("1, 2 эсвэл 3 гэж бич: ")
    if answer == "1" or answer == "2" or answer == "3":
        print("Баярлалаа, зөв тоо!")
        break
    print("Буруу байна, дахин оролд.")
📝 Компьютергүй сурагчид

«Зөв нууц үг хэлэх хүртэл хаалга онгойхгүй» тоглоом — давталтын логикийг биеэр тоглох.

✅ Дүгнэлт — өнөөдөр юу сурсан

while True: + break = зөв оролт авах хүртэл давтах. .isdigit() цифр эсэхийг шалгана, int() тоо болгоно.
