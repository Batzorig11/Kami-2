ЕРӨНХИЙ ХӨДӨЛГҮҮР — play_scene()

🎯 Зорилго

Ямар ч хэсгийн толь бичгийг хүлээн авч тоглуулдаг ганц функц бичих. Энэ нь өмнөх бүх давтагдсан кодыг устгана.

📘 Шинэ ойлголт

Хийсвэрлэл — «өгөгдөл» ба «логик»-ийг салгах. Нэг функц + олон өгөгдөл = олон хэсэг.

💡 Тайлбар

Энэ бол хамгийн чухал «аха» мөч. 6-7-р хичээлд бид хэсэг бүрийг гараар давтан бичсэн. Одоо бид нэг удаа ерөнхий play_scene(scene) функц бичнэ — энэ нь толь бичгээс гарчиг, түүх, сонголтыг уншиж, тоглогчоос асууж, зөв/бурууг шийднэ. Нэг функц + олон өгөгдөл = олон хэсэг. Үүнийг хийсвэрлэл гэнэ: «юу хийх» (логик) ба «ямар агуулга» (өгөгдөл) хоёрыг тусгаарлана. died() функцийг (үхлийн дэлгэц) мөн энд хамт бичнэ.

✍️ Дасгал (хичээл дээр)

  1. Багш 6-7-р хичээлийн давтагдсан кодыг сануулж: «Үүнийг нэг удаа бичье».
  2. Хамтдаа play_scene(scene) бичих — толь бичгээс уншиж, сонголт асууж, зөв/бурууг шийднэ.
  3. Нэг хэсгийн толь бичгийг өгч туршина.

💻 Гарах код — namjil.py

namjil.py 
def play_scene(scene):
    while True:
        header(scene["title"])
        for story_line in scene["story"]:
            slow(story_line)
        slow(scene["question"], "yellow", bold=True)

        choices = scene["choices"]
        for i in range(len(choices)):
            print(color("  [" + str(i + 1) + "] ", "white", bold=True) + choices[i]["text"])

        pick = ask_number(len(choices))
        chosen = choices[pick - 1]

        if chosen["good"]:
            slow(chosen["result"], "green")
            if chosen["learn"] != "":
                player["learned"].append(chosen["learn"])
            pause()
            return True
        else:
            slow(chosen["result"], "red")
            if died():
                continue          # дахин тоглоно
            else:
                return False
👩‍🏫 Багшид

Энэ бол хамгийн чухал «аха» мөч. Сурагчдад «нэг функц, олон өгөгдөл» гэдгийг дахин дахин онцол. died() функцийг (үхлийн дэлгэц) энд хамт бичнэ — box-оор гоёж, дахин оролдох эсэхийг асууна.

📝 Компьютергүй сурагчид

play_scene-ийг «дүрэм» болгон цаасан дээр алхам алхмаар бичих. Багш нэг толь бичиг «оруулахад» сурагчид функцийг гараар «ажиллуулна».

✅ Дүгнэлт — өнөөдөр юу сурсан

Нэг play_scene(scene) функц = ямар ч хэсгийг тоглуулна. Логик ба өгөгдлийг салгасан нь хийсвэрлэлийн гол хүч.
