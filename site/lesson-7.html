Функц — давталтыг багцлах

🎯 Зорилго

def-ээр функц үүсгэж, кодоо хэсэг бүрээр нэрлэж багцлах.

📘 Шинэ ойлголт

Функц тодорхойлох (def name():), дуудах (name()), toolbox.py-г анх танилцуулах (from toolbox import ...).

💡 Тайлбар

Функц бол нэр өгсөн кодын багц — нэг удаа бичээд олон удаа дуудаж болно. def scene_temple(): гэж тодорхойлоод, дараа нь scene_temple() гэж дуудна. Энэ нь давтагдсан кодыг цэгцлэх анхны алхам. Мөн өнөөдөр бид toolbox.py-г танилцуулна — багшаас бэлэн өгсөн туслах функцүүд (color, clear, slow, box, ask). Эдгээрийг from toolbox import ... гэж оруулаад, дотор нь яаж ажилладгийг мэдэхгүйгээр зүгээр л дуудна.

✍️ Дасгал (хичээл дээр)

  1. Багш toolbox.py-г тарааж, color, clear, slow, box-ыг үзүүлнэ.
  2. Хэсэг бүрийг функц болгох: def scene_temple():, def scene_bird():.
  3. main() функц бичиж дотроос нь дуудах. Хэвлэлээ slow(), color()-оор гоёх.

💻 Гарах код — namjil.py

namjil.py 
from toolbox import color, clear, slow, box, ask

def scene_temple():
    clear()
    box(["Хийдэд"])
    slow("Багш: «Замдаа бүгдийг ажигла» гэв.")
    slow("[1] Ажиглах   [2] Зүтгэх")
    choice = ask("Сонголт: ")
    if choice == "1":
        slow("Замд гарлаа!", "green")
    else:
        slow("Чи мөхөв.", "red")

def scene_bird():
    clear()
    box(["Хар шувуу"])
    # ... ижил бүтэц ...

def main():
    scene_temple()
    scene_bird()

main()
📝 Компьютергүй сурагчид

«Функц бол жор» зүйрлэл — «цай чанах» жорыг нэг удаа бичээд олон удаа «дуудах». Цаасан дээр өөрийн функц зохиох.

✅ Дүгнэлт — өнөөдөр юу сурсан

Функц = нэртэй кодын багц. def-ээр зарлаж, нэрээр дуудна. toolbox.py-аас бэлэн туслахуудыг дуудаж хэрэглэнэ.
