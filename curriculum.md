## Багшид зориулсан үндсэн зарчим

1. **Бэлэн «хэрэгслийн хайрцаг» (toolbox).** Өнгө, хүрээ, бичгийн машины эффект зэрэг **хэцүү** функцийг сурагчид өөрсдөө бичихгүй — зүгээр л `toolbox.py`-аас дуудаж хэрэглэнэ (яг `print()` шиг). Эдгээрийг 7-8-р хичээлд танилцуулна.
2. **Эхлээд тодорхой, сүүлд хийсвэр.** Сурагчид эхэндээ хэдэн хэсгийг `if/elif/else`-ээр **гараар, давтан** бичнэ. Давталт нь уйтгартай болсон хойно (11-р хичээл) бид нэг ерөнхий `play_scene()` функц бичиж бүх давталтыг устгана — энэ нь функц/толь бичгийн **ач холбогдлыг мэдрүүлэх** гол мөч.
3. **Хүндрэлийн оргилууд:** 11-р хичээл (хийсвэрлэл) ба 14-р хичээл (шөнийн тоо тааруулах логик). Эдгээрт нэмэлт цаг, тусламж нөөцлөнө.

### Компьютергүй сурагчдад зориулсан ерөнхий зөвлөмж
- **Хосоор ажиллах (pair programming):** Нэг компьютерт 2 сурагч. Нэг нь «жолооч» (бичнэ), нөгөө нь «навигатор» (унших, алдаа хайх). 20 минут тутамд үүрэг солино.
- **Цаасан код:** Кодыг цаасан/самбар дээр бичих, гараар «ажиллуулж» (trace) гаралтыг таамаглах.
- **Дүрд тоглох:** Үлгэрийн хэсгүүдийг ангид жүжиглэн тоглож, сонголт-үр дагаврыг биеэр мэдрэх.

# 5-р хичээл: while давталт — оролтыг шалгах

# 7-р хичээл: Функц — давталтыг багцлах

**Шинэ ойлголт:** Функц тодорхойлох (`def name():`), дуудах (`name()`), `toolbox.py`-г **анх танилцуулах** (`from toolbox import ...`).

**Дасгал (хичээл дээр):**
1. Багш `toolbox.py`-г тарааж, `color`, `clear`, `slow`, `box`-ыг үзүүлнэ.
2. Хэсэг бүрийг функц болгох: `def scene_temple():`, `def scene_bird():`.
3. `main()` функц бичиж дотроос нь дуудах. Хэвлэлээ `slow()`, `color()`-оор гоёх.

```python

def scene_bird():
    clear()
    box(["Хар шувуу"])
    # ... ижил бүтэц ...

def main():
    scene_temple()

main()
```

**Компьютергүй сурагчид:** «Функц бол жор» зүйрлэл — «цай чанах» жорыг нэг удаа бичээд олон удаа «дуудах». Цаасан дээр өөрийн функц зохиох.

# 8-р хичээл: Параметр ба буцаах утга (return)

**Зорилго:** Функцэд утга дамжуулж (параметр), функцээс утга буцаахыг (`return`) сурах.

**Шинэ ойлголт:** Параметр (`def f(x):`), `return`, буцаасан утгыг хувьсагчид авах.

**Дасгал (хичээл дээр):**
1. `ask_number(count)` функц бичих — 1..count хүртэлх зөв дугаар буцаана (5-р хичээлийн давталтыг функц болгох).
2. Функц `True`/`False` буцаадгийг туршиж үзэх.
3. `ask_number`-г хэсгүүддээ ашиглах.

**Гарах код — `namjil.py`-д нэмэх (эцсийн хувилбартай ижил):**
```python
def ask_number(count):
    """1-ээс count хүртэлх ЗӨВ дугаар авах хүртэл асууна."""
    while True:
        answer = ask("  Чиний сонголт (дугаар): ")
        if answer.isdigit():
            number = int(answer)
            if 1 <= number <= count:
                return number
        print(color("  1-ээс " + str(count) + " хүртэлх дугаар бичнэ үү.", "red"))
```

**Компьютергүй сурагчид:** «Машин» тоглоом — оролт өгөхөд гаралт буцаадаг хүн-машин. Параметр болон буцаах утгыг биеэр дүрслэх.

# 10-р хичээл: Толь бичиг (dict)

**Дасгал (хичээл дээр):**

3. `player = {"name":..., "learned":[], "stage":1}` төлвийг үүсгэх.

**Гарах код — нэг хэсгийн өгөгдөл:**
```python
scene = {

**Компьютергүй сурагчид:** «Хувийн хэрэг» зүйрлэл — нэр, нас, ангиар нь толь бичиг зурах. Түлхүүрээр утга «хайх» дасгал.

# 11-р хичээл: ЕРӨНХИЙ ХӨДӨЛГҮҮР — play_scene() ⭐ (хүндрэлийн оргил)

**Зорилго:** **Ямар ч** хэсгийн толь бичгийг хүлээн авч тоглуулдаг ганц функц бичих. Энэ нь өмнөх бүх давтагдсан кодыг устгана.

**Шинэ ойлголт:** Хийсвэрлэл — «өгөгдөл» ба «логик»-ийг салгах. Нэг функц + олон өгөгдөл = олон хэсэг.

**Дасгал (хичээл дээр):**
1. Багш 6-7-р хичээлийн давтагдсан кодыг сануулж: «Үүнийг **нэг удаа** бичье».
2. Хамтдаа `play_scene(scene)` бичих — толь бичгээс уншиж, сонголт асууж, зөв/бурууг шийднэ.
3. Нэг хэсгийн толь бичгийг өгч туршина.

**Гарах код — `namjil.py`-ийн зүрх (эцсийн хувилбар):**

```

> **Багшид:** Энэ бол хамгийн чухал «аха» мөч. Сурагчдад «нэг функц, олон өгөгдөл» гэдгийг дахин дахин онцол. `died()` функцийг (үхлийн дэлгэц) энд хамт бичнэ — `box`-оор гоёж, дахин оролдох эсэхийг асууна.

**Компьютергүй сурагчид:** `play_scene`-ийг «дүрэм» болгон цаасан дээр алхам алхмаар бичих. Багш нэг толь бичиг «оруулахад» сурагчид функцийг гараар «ажиллуулна».

# 12-р хичээл: SCENES жагсаалт + main давталт

**Зорилго:** Хэсгүүдийг өгөгдлийн жагсаалт болгож, `main()`-ээр дараалан тоглуулах.

**Шинэ ойлголт:** Толь бичгүүдийн жагсаалт (`SCENES = [ {...}, {...} ]`), жагсаалтыг `for`-оор дамжих, `player["stage"]` шинэчлэх.

**Дасгал (хичээл дээр):**
1. 1–4-р хэсгийг (Хийд, Шувуу, Зээр, Хулгана) толь бичиг болгож `SCENES`-д нэмэх.
2. `main()` бичиж бүх хэсгийг дараалан тоглуулах.
3. `play_scene` `False` буцаавал тоглоомыг зогсоох.

**Гарах код — `namjil.py`:**
```python
SCENES = [
    { "title": "1-р хэсэг: Хийдэд", ... },
    { "title": "2-р хэсэг: Хар шувуу", ... },
    { "title": "3-р хэсэг: Хоёр зээр", ... },
    { "title": "4-р хэсэг: Хоёр хулгана", ... },
]

def main():
    number = 1
    for scene in SCENES:
        player["stage"] = number
        if not play_scene(scene):
            print("Аялал тасарлаа.")
            return
        number = number + 1
    print("Чи яллаа!")
```

**Компьютергүй сурагчид:** Дөрвөн хэсгийн текстийг **зохиох** баг даалгавар — кодоо бичих сурагчдад «өгөгдөл» болгон өгнө (бичих хүн ба зохиох хүний хамтын ажил).

# 13-р хичээл: Толгой самбар + сурсан тарниуд (5–6 хэсэг)

**Зорилго:** Дэлгэцийн дээд хэсэгт явц болон сурсан тарниудыг харуулах. Хятад худалдаачин ба Хааны ордны хэсгийг нэмэх.

**Шинэ ойлголт:** `player` толь бичгийн өгөгдлийг дэлгэцэнд харуулах, `if len(...) > 0:`, давталтаар жагсаалт хэвлэх.

**Дасгал (хичээл дээр):**
1. `header(title)` функц бичих — гарчиг, `Аялал: x / 12`, сурсан тарниуд.
2. 5-р (Худалдаачин), 6-р (Ордон) хэсгийг `SCENES`-д нэмэх.
3. Худалдаачны хэсэг `learn: "Хятадын гар дохио"` өгдгийг анзаарах — энэ хожим хэрэгтэй!

**Гарах код — `header`:**
```python
def header(title):
    clear()
    box([title], "magenta", center=True)
    print(color("  Аялал: " + str(player["stage"]) + " / 12", "grey"))
    if len(player["learned"]) > 0:
        print(color("  Ажигласан тарни:", "cyan"))
        for tarni in player["learned"]:
            print(color("    • " + tarni, "cyan"))
    print()
```

**Компьютергүй сурагчид:** «Аяллын дэвтэр» — сурагч бүр өөрийн ажигласан тарниудын самбарыг цаасан дээр хөтөлж, хэсэг бүрт шинэчилнэ.

# 14-р хичээл: Шөнийн оньсого (7–9 хэсэг) ⭐ (хүндрэлийн оргил)

**Зорилго:** Гурван шөнийн тагнуулын хэсгийг бүтээх. Энд **логик оньсого** бий: гадаа байгаа хүний **тоонд** тохирох тарниг сонгоно (1 хүн → шувуу, 2 хүн → зээр/хулгана).

**Шинэ ойлголт:** Шинэ синтакс байхгүй — өмнөх бүх ойлголтыг **логик бодлого** болгон нэгтгэх. Сонголт-үр дагаврыг утга учиртай зохиох.

**Дасгал (хичээл дээр):**
1. 7-р хэсэг (НЭГ тагнуул → нэг шувуу зөв) зохиох.
2. 8, 9-р хэсэг (ХОЁР тагнуул → хоёр зээр / хоёр хулгана зөв) зохиох.
3. Яагаад тоо тааруулах нь зөв болохыг ангиар ярилцах (үлгэрийн гол заль).

**Гарах код — 7-р хэсгийн өгөгдөл (загвар):**
```python
{
    "title": "7-р хэсэг: Нэгдүгээр шөнө — НЭГ тагнуул",
    "story": [
        "Хаан ганц түшмэлээ тагнуулд илгээв.",
        "( Гадаа НЭГ хүн байна. )",
    ],
    "question": "Гадах хүний ТООНД тааруулж аль тарниа уншуулах вэ?",
    "choices": [
        {"text": "«Бараан шувуу вааг вааг» (нэг шувуу).",
         "good": True, "learn": "",
         "result": "Айсан түшмэл «намайг л хэлж байна» гэж зугтав."},
        {"text": "«Хоёр зээр жоорон жоорон» (хоёр зээр).",
         "good": False, "learn": "",
         "result": "Нэг хүнд «хоёр» гэсэн нь буруу. Хаан цаазлав."},
    ],
}
```

> **Багшид:** Энэ хичээл агуулгаараа хүнд. Сурагчдад «гадаа хэдэн хүн байна? → тэр тоотой тарни» гэсэн дүрмийг самбар дээр зурж тайлбарла. Зохиох ажлыг багаар хувааж болно.

**Компьютергүй сурагчид:** Гурван шөнийг жүжиглэх — «тагнуулчид» гадаа сонсож, Намжил зөв тоог сонгоход зугтана. Логикийг биеэр мэдрүүлнэ.

# 15-р хичээл: Төгсгөлийн хэсгүүд (10–12) + дэлгэцүүд

**Зорилго:** Бөгж олох, Ном хаялцах, Сүүлчийн царцааны хэсгийг бүтээж, эхлэл/ялалт/ялагдлын дэлгэцийг нэмэх.

**Шинэ ойлголт:** Шинэ синтакс байхгүй — бүтээлээ дуусгах. Үлгэрийн зангилаа (хулгайч, царцааны заль) кодоор илэрхийлэх.

**Дасгал (хичээл дээр):**
1. 10-р (Шорвог зутан → хулгайч бөгж өгөх), 11-р (Гурван лам → гар дохио), 12-р (Царцаа) хэсгийг нэмэх.
2. `intro()` ба `show_win()` дэлгэцийг `box`-оор бүтээх.
3. 12-р хэсгийн «заль» (нэр нь Царцаа, царцаатай таарах) яагаад инээдтэйг ярилцах.

**Гарах код — `show_win()`:**
```python
def show_win():
    clear()
    box([
        color("Ц А Р Ц А А   Н А М Ж И Л   Я Л Л А А !", "green", bold=True),
        "",
        "Ажигласан болгоноо цаг тухайд нь хэрэглэж, «мэргэн лам»",
        "хэмээх алдрыг олж, нутагтаа эсэн мэнд харьжээ.",
    ], "green", center=True)
```

**Компьютергүй сурагчид:** Үлгэрийн төгсгөлийг зурагт хуудас (комик) болгон зурах. Ялалтын дэлгэцийн текстийг хамтран зохиох.

# 16-р хичээл: Бүрэн туршилт, алдаа засах, ТОГЛООМЫН ӨДӨР

**Зорилго:** Тоглоомыг эхнээс нь дуустал тоглож, алдаа засаж, өөрийн санаа нэмж, бусдад үзүүлэх.

**Шинэ ойлголт:** Дибаг (алдаа олж засах), туршилт, кодоо сайжруулах. Шинэ синтакс байхгүй.

**Дасгал (хичээл дээр):**
1. **Бүрэн тоглолт:** Зөв замаар ялах, буруу сонголтоор үхэх — бүгдийг турших.
2. **Алдаа засах:** Таслал, хаалт, түлхүүрийн алдаа хайж засах (нийтлэг алдаануудын жагсаалтыг багш бэлдэнэ).
3. **Бүтээлч нэмэлт (сонголтоор):** Өөрийн шинэ хэсэг (`SCENES`-д нэг толь бичиг) зохиож нэмэх, эсвэл текст/өнгө сайжруулах.
4. **Тоглоомын өдөр:** Хосууд бие биеийнхээ тоглоомыг тоглож, санал солилцоно.

**Эцсийн үр дүн:** Бүрэн ажиллагаатай `namjil.py` (12 хэсэг) + `toolbox.py`. Сурагч өөрийн бичсэн кодын мөр бүрийг ойлгодог.

**Компьютергүй сурагчид:** «Чанарын шалгагч» үүрэг — тоглоомыг тоглож, алдаа/сайжруулах санааг бүртгэх. Шинэ хэсгийн санаа цаасан дээр зохиож, бичигч сурагчид өгөх.
